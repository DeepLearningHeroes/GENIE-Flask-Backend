# GENIE-Flask-Backend

### To create a new requirements.txt file - 
```pip freeze > requirements.txt``` 

### To setup locally - 
Run - 
<ol>
    <li>pip install -r requirements.txt</li>
    <li>.venv\Scripts\activate</li>
    <li>pymon app.py</li>
</ol>
